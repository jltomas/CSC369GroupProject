This project runs using the provided setup from the 369 lecture slides. 
It can be ran in one of two ways:

* Recommended * 
- Copy the folder 'cluster' to the ambari-head head server.
- Edit the 'run' file to reference your local directory instead of 'kheynema'.
- Type './run' inside the folder to run the job. 

* Alternative *
- The folder 'intellij' is set up to run the code locally using the provided idea project.
