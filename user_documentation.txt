- This project runs using the provided setup from the 369 lecture slides. 

- The Driver.scala file is set up to run in the provided IntelliJ project. 

- To run the file on the cluster, change the code to remove the SetMaster() call. 

- The Driver.scala file has different defined methods for each part of the analysis.

- To see the results from one particular part, change the Driver.scala file to call that particular method.
